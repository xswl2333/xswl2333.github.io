AABB包围盒和OBB包围盒

AABB 包围盒碰撞检测

适用于快速初步碰撞检测（Broad Phase）。

二维空间中，AABB 是一个矩形；三维空间中，AABB 是一个立方体。

不支持旋转，因此对复杂形状或旋转物体的检测精度较低。

碰撞检测条件：

两个 AABB 相交的条件是：在所有轴上，它们的边界都有重叠。

具体条件如下：

  • 在 X 轴上：A.maxX ≥ B.minX 且 A.minX ≤ B.maxX
  • 在 Y 轴上：A.maxY ≥ B.minY 且 A.minY ≤ B.maxY
  • 在 Z 轴上（三维空间）：A.maxZ ≥ B.minZ 且 A.minZ ≤ B.maxZ

OBB 包围盒碰撞检测

分离轴定理（SAT）：通过投影到多个分离轴上，判断两个 OBB 是否相交

具体步骤：

  1. 计算两个 OBB 的中心点和轴向。
  2. 将每个 OBB 的顶点投影到这些轴上。
  3. 检查投影区间是否有重叠。如果所有轴上的投影都有重叠，则两个 OBB 相交

判断点是否在三角形内

AP,BP,CP分别叉乘,之后点乘判断方向,即点乘为正值,点p在三角形内部

求解一个向量在另一个向量的投影

a · b = ||a|| ||b|| cosθ 点乘公式

首先计算向量夹角 θ = arccos[(a·b)/(||a|| ||b||)]

然后可得 proj = b cosθ

简化公式可得

如果 b 是单位向量，计算可简化为：proj = (a · b) * b

已知入射向量和法线向量求解反射向量

向量加减法,向量投影法

ON = 2S (S为法向量的一半)

R = I + 2S (画图可知,向量加法)

因为S 是 -I在N的投影

S = - (I · N)N

所有 R=I−2(I⋅N)N

求解一个向量在一个平面上投影

计算向量 v 在法向量 n 上的投影向量 vproj-n

从向量 v 中减去 vproj-n，得到向量 v 在平面上的投影向量 vproj

如果法向量 n 是单位向量，即其长度为 1，则 n⋅n=1，公式简化为：

v_proj = v - (n·v)n

判断一个点是否在平面之上

点法式平面方程：n·(P - P₀) = 0

  • n 是平面法向量（需归一化）
  • P₀ 是平面上已知点
  • P 是待测试点

判断一个点在是否在视锥之内

n·(P - P₀) >= 0 则说明点和法向量方向相同

然后依次判断六个面,符合条件则说明在视锥之内
