pyenv

pyenv 是一个用于管理多个 Python 版本的工具,可以灵活切换不同版本。

常用命令速查
命令 描述
pyenv install --list 列出所有可安装的 Python 版本
pyenv install <version> 安装指定版本的 Python (如 pyenv install 3.11.6)
pyenv versions 列出当前已安装的所有版本，并标记当前激活的版本
pyenv global <version> 设置全局默认 Python 版本
pyenv local <version> 为当前目录设置局部 Python 版本（生成 .python-version 文件）
pyenv shell <version> 仅在当前 Shell 会话中切换版本
pyenv uninstall <version> 卸载指定版本的 Python
pyenv rehash 重新生成 shim 文件（通常在安装新包或新版本后使用）
